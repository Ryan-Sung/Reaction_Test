# Reaction_Test

## 實驗簡述：
實驗為接尺實驗的改造版，共有三組變因：
1. (A組) 對照組(原版接尺實驗)
2. (B組) 99 乘法組(邊背誦 99 乘法邊反應接尺)
3. (C組) 拍肩舉手組(一旁有第三者不定時拍肩，被拍肩時需舉起另一隻手)

全班 41 人，每人每組變因進行 5 次
用最小刻度 1mm 的直尺測量「反應尺落下至接住尺所需距離」
